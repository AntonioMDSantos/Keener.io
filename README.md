# Keener.io

#### O que foi utilizado:

```
node, nodemon, sequelize, express, mysql
```

### Para iniciar o programa, primeiro abra na pasta backend
```
cd backend
```

### Após isso, inicie o nodemon com o comando
```
yarn dev
```

### ```diff
